# اپلیکیشن محاسبه قیمت کرکره برقی

این اپلیکیشن با **Kotlin + Jetpack Compose + Room** ساخته شده و قابلیت محاسبه قیمت کرکره، موتور، شفت، قوطی و امکانات بیشتر را دارد. همچنین می‌توانید گزارشات مشتریان را ذخیره و مشاهده کنید.

---

## **ویژگی‌ها**

- محاسبه خودکار **مساحت کرکره** و قیمت نهایی  
- انتخاب **تیغه، موتور، شفت، قوطی** و **امکانات بیشتر**  
- مدیریت **هزینه‌های اجرایی** (نصب، جوشکاری، کرایه حمل)  
- ذخیره محاسبات هر مشتری با تاریخ و مشاهده/حذف آن‌ها  
- جستجوی نام مشتری در گزارشات  
- دارک تم حرفه‌ای و رابط کاربری مدرن

---

## **صفحات اپلیکیشن**

1. **صفحه اصلی:** محاسبه قیمت کرکره و مدیریت انتخاب‌ها  
2. **صفحه قیمت پایه:** افزودن/ویرایش/حذف تیغه، موتور، شفت، قوطی و امکانات بیشتر  
3. **صفحه گزارشات:** مشاهده و حذف محاسبات مشتریان  

> ⚠️ صفحه **قطر رول** بعداً اضافه خواهد شد.

---

## **نصب و اجرا**

1. پروژه را روی **Android Studio** باز کنید  
2. مطمئن شوید **Gradle Sync** انجام شده است  
3. با استفاده از **Build > Build Bundle(s) / APK(s) > Build APK(s)** خروجی APK را بسازید
